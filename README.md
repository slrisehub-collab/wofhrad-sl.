## 🚀 Deployment (GitHub Pages)

This is a **zero-dependency, single HTML file** — no build step needed.

### Steps to deploy:

1. Push `index.html` to the `main` branch of your GitHub repository.
2. Go to your repository → **Settings** → **Pages**.
3. Under **Source**, select **Deploy from a branch** → branch: `main`, folder: `/ (root)`.
4. Click **Save**. Your site will be live in ~1 minute at:
   `https://<your-username>.github.io/<repo-name>/`

## ✏️ How to Edit

Open `index.html` in any text editor. Search for the section you want to update:

- **Email** → search `info@wofhrad.com`

- **Programs** → search `id="programs"`
